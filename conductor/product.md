# Product Guide

## Initial Concept

A tool license management system that controls user access to software tools with device-based limitations.

## Vision

A self-hosted license management server that enables software creators to protect and monetize their tools by controlling user access through flexible licensing models and device-based restrictions.

## Target Users

- **Non-technical end users** who purchase and use licensed software tools
- **Tool creators/vendors** who manage licenses through the admin interface

## Core Features

### License Models
- **Per-device limit** - Restrict activation to a maximum number of devices
- **Time-based expiration** - Licenses valid for a defined period
- **Subscription** - Recurring payment with auto-renewal support
- **One-time purchase** - Perpetual lifetime licenses

### Device Identification
- **Hardware fingerprint** - CPU, motherboard, disk serial numbers
- **MAC address** - Network adapter identifier
- Combined approach for robust device tracking

### Admin Features
- **Dashboard** - View all licenses, users, and device activations
- **License generation** - Create and issue license keys
- **Remote deactivation** - Revoke device access remotely
- **Usage analytics** - Track tool usage patterns

## Deployment

- **Self-hosted server** - Full control over infrastructure and data
