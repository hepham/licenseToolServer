# Tech Stack

## Backend

- **Framework:** Django 5.x
- **Language:** Python 3.11+
- **API:** Django REST Framework (DRF)
- **Authentication:** JWT (via djangorestframework-simplejwt)

## Database

- **Primary:** MySQL 8.x / MariaDB 10.x
- **ORM:** Django ORM

## Frontend

- **Framework:** React 18.x
- **Build Tool:** Vite
- **State Management:** React Context / Redux Toolkit
- **HTTP Client:** Axios
- **UI Library:** Tailwind CSS

## Infrastructure

- **Containerization:** Docker + Docker Compose
- **Caching:** Redis
- **Task Queue:** Celery (with Redis broker)
- **Web Server:** Nginx (reverse proxy)
- **WSGI:** Gunicorn

## Development Tools

- **Package Manager:** pip + requirements.txt (backend), npm (frontend)
- **Linting:** flake8, black (Python), ESLint, Prettier (JS)
- **Testing:** pytest (backend), Jest (frontend)

## Device Fingerprinting Libraries

- Hardware ID collection (CPU, disk, motherboard serials)
- MAC address detection
- Client SDK for tool integration
